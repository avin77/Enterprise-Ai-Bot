    3. Define TraceEvent dataclass (or dict, your choice):
       - event_type: str = "agent_trace"
       - session_id: str
       - turn_id: str (unique turn identifier, e.g. UUID)
       - intent: str
       - intent_confidence: float
       - routing_target: str (retrieval|tool|fallback)
       - retrieved_doc_ids: list[str] (source document IDs)
       - llm_prompt_tokens: int (cumulative across all agent calls)
       - llm_response_tokens: int
       - llm_latency_ms: int (total LLM time, not including retrieval)
       - tool_calls: list[dict] (tool name and result, no sensitive data)
       - total_latency_ms: int (full turn time)
       - grounded: bool (response cites sources — determined by pattern check or LLM judge)
       - grounding_signal: str (e.g., "has_source_attribution", "no_sources", "ambiguous")
       - fallback_triggered: bool
       - timestamp: str (ISO 8601 with Z)

    4. Implement emit_trace_event() function:
       ```
       async def emit_trace_event(
           session_id: str,
           turn_id: str,
           intent: str,
           intent_confidence: float,
           routing_target: str,
           retrieved_doc_ids: list[str],
           llm_prompt_tokens: int,
           llm_response_tokens: int,
           llm_latency_ms: int,
           tool_calls: list[dict],
           total_latency_ms: int,
           fallback_triggered: bool,
           grounded: bool = True,
           grounding_signal: str = "unknown"
       ) -> None:
       ```

    5. Implementation:
       - Build TraceEvent with current timestamp
       - Convert to JSON via asdict() or dataclass-to-dict
       - Log via logger.info(json.dumps(event))

    6. Error handling:
       - If JSON serialization fails, log the error but don't crash
       - If logger fails, swallow the error (logging is best-effort)

    ```
    fields @timestamp, intent, intent_confidence, routing_target, fallback_triggered
    | filter event_type = "agent_trace"
    | stats count() by intent
    | sort count desc
    ```
