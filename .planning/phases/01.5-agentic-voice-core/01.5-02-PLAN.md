<task type="auto">
  <name>Task 2: Create ResponseAgent (Claude sonnet synthesis)</name>
  <files>backend/app/agents/response.py</files>
  <action>
    Create ResponseAgent class for grounded synthesis:

    1. Constructor: receive `aws_clients: AwsClientBundle`
    2. Lazy-load Bedrock client via aws_clients.bedrock()
    3. Use model ID: `anthropic.claude-sonnet-4-20250514-v1:0` (from 01.5-RESEARCH.md)
    4. Temperature = 0 (consistent with orchestrator for reproducible evals)
    5. max_tokens = 300 (voice-friendly, hard limit per 01.5-RESEARCH.md)

    Method signature:
    ```
    async def synthesize(
        self,
        query: str,
        chunks: list[ChunkResult],
        history: list[Message] = None
    ) -> str:
    ```

    System prompt construction:
    - Base grounding rules (from 01.5-RESEARCH.md):
      1. ONLY state facts from provided context chunks
      2. EVERY factual claim followed by source: "According to [source_doc]..."
      3. If no context answers question: "I don't have that specific information. You can contact Jackson County at 816-881-3000."
      4. No bullet points/numbered lists for voice
      5. Keep under 3 sentences for voice delivery
    - Import guardrails from Phase 2: `build_response_system_prompt()`
    - Merge Phase 2 guardrails into the system prompt

    Implementation:
    - If history is None, use empty list
    - Format chunks as: "[CHUNK: {source_doc} | {text}]" (per 01.5-RESEARCH.md)
    - Build messages: history[-5:] + user message with chunks (5-turn window)
    - Call bedrock.converse() with system prompt + messages
    - Parse response text
    - Return response text

    Error handling:
    - If Bedrock call fails: log error, return "I'm having trouble processing that request. Please try again."
    - If chunks is empty: still call Claude but with system prompt noting no context available

    Imports:
    - from backend.app.agents.types import Message
    - from backend.app.agents.retrieval import ChunkResult
    - from backend.app.safety.guardrails import build_response_system_prompt
    - from backend.app.services.aws_clients import AwsClientBundle
    - boto3, logging, asyncio
  </action>
