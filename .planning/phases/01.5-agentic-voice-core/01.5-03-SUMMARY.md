## Decisions Made

- `ConversationTurn` includes agent metadata fields (intent, confidence, routing_target) not just user/assistant text — enables DynamoDB writes with enriched data from day one
- DynamoDB SK uses `int(hashlib.md5(session_id.encode()).hexdigest(), 16) % 10` prefix — prevents hot partitions under high concurrent load
- `MockToolAgent.latency_ms=50` is hardcoded to simulate realistic tool call timing; Phase 5 RealToolAgent will measure real API latency
